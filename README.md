# 📝 Task Manager CLI

Консольный **todo list на Java** с сохранением данных в файл.
Проект создан для закрепления ключевых тем: коллекции, работа с файлами, исключения и базовая архитектура приложения.

---

## 🚀 Возможности

* ➕ Добавление задач
* 📋 Просмотр списка задач
* ❌ Удаление задач
* ✔ Отметка задачи как выполненной
* 💾 Сохранение задач в файл
* 🔄 Загрузка задач при запуске

---

## 🧱 Архитектура

Проект построен по принципу разделения слоёв:

```
model/        // модель данных (Task)
service/      // бизнес-логика (TaskService, FileService)
ui/           // пользовательский интерфейс (Console)
Main.java     // точка входа
```

---

## 🛠 Технологии

* Java Core
* Collections
* File I/O (BufferedReader, BufferedWriter)
* Exception Handling

---

## 💾 Формат хранения

Задачи сохраняются в файл `tasks.txt` в читаемом формате:

```
[✔] Сходить в магазин
[ ] Сделать домашку
```

---

## ▶️ Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/mevenzy/file-backed-task-manager.git
cd file-backed-task-manager
```

### 2. Скомпилировать и запустить

```bash
javac Main.java
java Main
```
